About This Project

This project is a Credit Approval Predictor that uses Machine Learning to determine whether a user will be approved for credit based on key financial and demographic features.

It considers attributes such as age, job type, marital status, education level, account balance, housing loan, and past campaign interaction details.

The model is trained on a real banking dataset and can be used to help banks and financial institutions assess creditworthiness.

Built using Flask, Python, and Bootstrap, this tool provides an easy-to-use web interface to input data and receive predictions in real-time.
